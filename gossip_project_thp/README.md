THPotin/The Gossip Project/...

ON A FAIS LES BONUS, BITCH


Pour faire fonctionner tout ça rien de bien compliqué : 
~~~~~~~~~~~~~~~~~~~~
$ bundle install

$ rails db:migrate

$ rails db:seed

$ rails c

$ tp User.all
